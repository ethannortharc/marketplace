# Workbench Plugins

Personal Claude Code plugin marketplace.

## Plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| [ratchet](https://github.com/ethannortharc/ratchet) | 5.1.0 | Intent-driven autonomous execution with multi-perspective story alignment, sprint planning, and ratchet optimization |

## Install

In Claude Code:

```
/plugin marketplace add ethannortharc/marketplace
/plugin install ratchet@ethannortharc-marketplace
```

Choose "Install for you (user scope)" for global availability.
