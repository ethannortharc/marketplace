# Workbench Plugins

Personal Claude Code plugin marketplace.

## Plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| [ratchet](https://github.com/ethannortharc/ratchet) | 6.0.0 | Multi-agent collaboration platform with Python process management, dual-track Story/Sprint model, SQLite state, and regression testing |

## Install

In Claude Code:

```
/plugin marketplace add ethannortharc/marketplace
/plugin install ratchet@ethannortharc-marketplace
```

Choose "Install for you (user scope)" for global availability.
